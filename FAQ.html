Frequently Asked Questions

Find answers to common questions below.


What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of webpages.

What is CSS?

CSS stands for Cascading Style Sheets. It is used to style webpages.

What is JavaScript?

JavaScript adds functionality and interaction to webpages.

Why do we use forms?

Forms are used to collect information from users.
